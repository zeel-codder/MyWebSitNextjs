## Sept 3 : Write Code

Create one .py file and copy pase code in it.

```python
try:
    from PIL import Image
except ImportError:
    import Image
import pytesseract


f1=open('output.txt','w')
f1.write((pytesseract.image_to_string(Image.open('index.png'))))
```

And By Running this code it will convert or Extract Text of index.png file and put the text in output.txt file.
