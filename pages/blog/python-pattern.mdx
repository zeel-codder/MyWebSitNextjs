## P.3. Pascal.

*Program:*

```python
def pattern(n):
    for i in range(1,n+1):
        for j in range(1,i+1):
            print(f(i-1,j-1)," ",end='')
        print('\r')
def f(n,i):
    res=1
    if(i>n-i):
        i=n-i
    for k in range(0,i):
        res=res*(n-k)
        res=res//(k+1)
    return res
s=int(input('Enter number of row:\n'))
```

## P.12. Fibonacci.

*Program:*

```python
def function():
    f,s=1,1
    while True:
        yield f
        f,s=s,f+s
for x in function():
    if(x>50):
        break
    print(x,end=" ")
```
